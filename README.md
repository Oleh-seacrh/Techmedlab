# Techmed Lab Chemicals Trading

One-page landing website for Techmed Lab — quality medical chemicals and supplies.

## Tech Stack

- **Next.js 15** (App Router)
- **Tailwind CSS**
- **Framer Motion**
- **TypeScript**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Placeholders

Replace or add as needed:

- **Logo** — Header currently uses text "Techmed Lab"
- **Hero background** — Add imagery via `Hero.tsx` or `globals.css`
- **Brand logos** — `Brands.tsx` shows text; add SVG/image logos
- **Contact form** — Wire to your backend, Formspree, or similar
